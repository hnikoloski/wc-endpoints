# Endpoints

```
/wp-json/dp-api/v1/
```

Get All Products

```
/wp-json/dp-api/v1/products
```

Get Single Product

```
/wp-json/dp-api/v1/product/<id>
```

Get All Categories

```
/products/categories
```

Get Single category

```
/products/category/<id>
```

Get user Info

```
/wp-json/dp-api/v1/userinfo/<id>
```
